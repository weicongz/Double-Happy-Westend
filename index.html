GitHub 访问令牌

如何获取GitHub访问令牌：

  1. 登录您的GitHub账号
  2. 访问 Personal Access Tokens
  3. 点击"Generate new token"
  4. 设置描述（例如："商品管理系统"）
  5. 选择"gist"权限
  6. 点击"Generate token"
  7. 复制生成的令牌并粘贴到上方输入框
